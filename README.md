EDA Visualizer is a powerful tool designed to help users explore and analyze their datasets in a visually intuitive way. The main objective is to provide an in-depth analysis of the dataset. 
The focus are on three aspects: Data Visualization, Statistical Analysis and Exploratory Data Analysis (EDA). 
The product will offer a comprehensive tools and features to help users understand their data and make informed decisions based on the insights gained from the analysis.
The following features are included in the product:

-> Visualizations
 Histograms
 Box Plots
 Scatter Plots
 Correlation Heatmaps
-> Statistical Analysis
 Summary Statistics
 Measures of Central Tendency
 Measures of Dispersion
-> Exploratory Data Analysis
 Missing Value Analysis
 Outlier Analysis
